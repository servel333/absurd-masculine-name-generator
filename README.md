Absurd Masculine Name Generator
===============================

Prints names that are masculine and absurd.

## To run

* Install Ruby
* Run the script

```
ruby ruby absurd-masculine-name-generator.rb
```

## Some examples

* Meat McSteveson
* Bullock McDukearm
* Stag Beeflion
* Buck McHammerleg
* Sir Lionhammer
* Mr. McMeatarm
